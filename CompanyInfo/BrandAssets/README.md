# Brand Assets

Primary logo source files for Smart Business Valee website.

- `public/brand/logo-primary.svg` : Main horizontal logo (website header/footer)
- `public/brand/logo-icon.svg` : Icon-only mark (social/profile/favicon base)

Usage guidance:

- Prefer SVG files for all web usage.
- Keep aspect ratio locked when resizing.
- Do not alter icon flow (brain -> briefcase -> dollar).
- Do not recolor outside the defined gold + charcoal palette unless creating approved alternates.
- `public/brand/logo-primary-dark.svg` : Dark-background blend variant for navbar/footer
